## 0. 战略相关性（Strategic Context · ≤3 行）

劳保展（CIOSH）作为 MDS 在中国的成熟收购项目，已进入瓶颈/萎缩期。
需要通过"第二曲线"为展会创造新的落脚点与架构。

## 2. 核心问题（Root Issue）

> **展商品类过度单一**，全部集中在**低级别 PPE 产品**：
> 手套 / 劳保服 / 防护面具 / 面料 等。
>
> 这是"全国最大"带来的同时也是"无法继续增长"的根因。

## 3. 组织架构（Org Structure）

### MDS 团队（5 人）
- 项目执行 ×2
- 市场部 ×1
- 全职销售 ×1
- 项目总监 ×1

### 中方团队（7 人）
- 架构与 MDS 类似（执行 / 市场 / 销售 / 总监结构）

### 协作机制
- **项目执行**与**招商**：双方分开运作，最终汇总到合资公司层面
- **Budget**:双方分开管理，按**年度**进行汇总结算
- 合资公司是结果汇聚层,不是日常执行层

## 4. LTC 现状（Lead-to-Cash）

### 4.1 总体特征
项目处于稳定/瓶颈期。PPE 体量大、受众广,**当前盘面不缺线索**。
中方的协会会员资源已足够支撑现有招商需求。

### 4.2 线索来源现状

| 渠道 | 现状 | 问题 |
|---|---|---|
| 中方协会会员资源 | 主要来源,稳定 | 同质化,无法带来品类突破 |
| 垂直展会走访(风电/消防/户外用品) | 销售定期走访 | **品类断层**:垂直展的高端产品 vs 劳保展的低端产品,无法对接 |
| 百度推广 | 在做 | 能力薄弱 |
| EDM | 在做 | 能力薄弱 |
| 公众号推文 | 在做 | 能力薄弱 |
| 领英 / Google 广告(海外) | 在做 | ROI 几乎为零 |
| 联合展整合型营销 | **无** | 缺失 |

### 4.3 关键断层(必须记住)
> 垂直展会(如风电)上的**高端防坠落**产品 ≠ 劳保展上的**低端防坠落**产品。
> 这意味着:**单纯走访垂直展无法直接转化展商**,需要新的对接机制。
