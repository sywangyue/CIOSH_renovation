# document_rules

## 命名规则

`YYYY-MM-DD_S[N]_[主题简述].md`

## 语言

English

## 示例

```
2026-04-24_S01_new_catalogs.md
2026-06-10_S02_visitor prompt design.md
2026-08-01_S03_exhibitor promot design.md
```

## 汇总文件（由 BD 总监操作）

每 1-2 个月由业主方封装一次，命名为：`YYYY-QN_summary.md`

## 原则

- 每份 S 文档只记录当次讨论，不溯改，不追加
- S 文档之间不互相引用，独立成册
- 汇总时由业主方决定哪些内容可以回写到 `Agent.md`（事实类才可回写）
- 推断、计划、假设：只进 S 文档，不进 `Agent.md`
