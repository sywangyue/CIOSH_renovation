### 各层详情

#### 第一层：B2C 免费平台

- 消费者访问 `app.sommelier.bot` → AI 对话推荐红酒
- 所有入驻商户的产品都在这个池子里
- 消费者得到推荐 → 跳转商户网站购买
- **收入模式**：不抽佣金，纯引流 → 商户看到流量价值 → 付费升级

#### 第二层：商户独立实例（€299/月）

- 商户上传产品库存（CSV/XML/Vivino feed）
- AI 自动富化：30+ 属性标签（口感、酒体、单宁、适饮期、风土、配餐）
- 品牌定制：颜色、语调、logo
- 多种入口形式：
  - **AI Banners**：页面上下文感知的智能横幅
  - **Interactive Chat Window**：可嵌入的对话窗口
  - **Interactive Wine Map**：5000+ 产区可视化探索
- 对话洞察 Dashboard：用户问了什么、趋势、热门产品
- 跨会话记忆：记得用户的对话历史、购买记录、口味偏好

#### 第三层：Enterprise 定制

- 对接 ERP/CRM 系统
- 人工客服转接
- 专用入驻支持和培训

## 三、商户入驻流程（四步）

| 步骤 | Sommelier.bot | 技术实现 | CIOSH 对标 |
|------|-------------|---------|-----------|
| **1. 上传库存** | CSV / Google Shopping XML / Vivino feed | 文件解析管道 | 展商上传产品清单（CSV/Excel/表单） |
| **2. AI 富化** | 30+ 属性自动标签（口感图谱、配餐等） | LLM + 向量化 | Skill 自动标注品类/产地/认证/产能 |
| **3. 品牌定制** | 颜色、语调、logo | 配置模板 | Personality + AGENTS.md 品牌定制 |
| **4. 粘贴 JS** | 一行 `<script>` 嵌入网站 | JS Widget | H5 Chat Widget URL 或 JS 嵌入 |

**关键洞察**：整个过程不需要商户写一行代码。第 4 步就是粘贴一段 JavaScript。

## 七、Sommelier.bot 的三点核心启示

### 启示 1：B2C 免费是获客引擎，不是产品

他们的 B2C 平台不赚钱。它的唯一目的是让消费者用起来 → 给商户带流量 → 商户看到价值 → 付费。

**CIOSH 同理**：你的公开 H5 Chat 不赚钱，但它让 4 万观众用起来 → 展商看到咨询量 → 展商愿意付费升级专属 Agent。

### 启示 2："一行 JS" 是这个品类的标准做法

不是 App、不是小程序、不是客户端。就是一行 `<script>`。因为 B2B 展商最怕"技术门槛"。一行 JS 贴在自己的官网上，就有 AI 客服了——这才是 B2B 能接受的部署复杂度。

### 启示 3：AI 富化数据是核心护城河

40 个商户都在用，不是因为他们的 AI 模型多好，而是因为他们把烂数据变成了好数据。30+ 属性标签让 AI 能真正"理解"一瓶酒——产地、口感、配餐、适饮期。

**CIOSH 的护城河同理**：你帮展商把"公司名+展位号"变成"品类+产地+认证+产能+对标客户"，这个结构化数据的价值远超 H5 Chat 本身。

## 八、差距评估

| 能力 | Sommelier.bot 已有 | CIOSH 现状 | 差距 |
|------|-------------------|-----------|------|
| 公开 AI 对话 | ✅ app.sommelier.bot | ❌ 无 | 需搭建 H5 Chat |
| 商户数据上传 | ✅ CSV/XML 导入 | ✅ exhibitors.csv | 需结构化管道 |
| AI 数据富化 | ✅ 30+ 属性自动标签 | ❌ 无 | 需开发 Skill |
| 商户独立实例 | ✅ €299/月 | ❌ 无 | 需展商 Profile 模板 |
| JS 嵌入 | ✅ 一行 `<script>` | ❌ 无 | 需 Chat Widget |
| 对话洞察 | ✅ Dashboard | ❌ 无 | 不急，可后做 |
| 跨会话记忆 | ✅ 用户画像 | ✅ Hermes Memory | 已完成 |
| 多平台自动采集 | ❌ 无 | ✅ OpenCLI 自媒体扫描 | **反超** |
| 主动信号捕获 | ❌ 纯被动等用户来 | ✅ S07 已验证可行性 | **反超** |

**关键发现**：Sommelier.bot 在"被动服务"（等用户来问）方面成熟，但在"主动获客"（去自媒体找需求）方面是零。**这是 CIOSH 的差异化武器。**
