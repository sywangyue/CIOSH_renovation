**API 通信**：

- 打开 Network 面板，刷新页面，筛选 XHR / Fetch 请求
- 记录前 5 个接口请求的：URL 路径格式 / HTTP 方法（GET/POST）/ 返回数据格式（JSON/XML）

**宿主环境**：

- 检查 Console 里有无报错信息
- 检查页面 title 和 favicon，确认这是「劳保云」实例还是展商秀通用控制台

### 2.5 已知问题 / 异常记录

1. **展会已结束**：当前调研账号绑定的是历史展会（2020年自行车展 + CIOSH 劳保展），状态显示「展会已结束」，导致询盘、直播间、联络人等模块无数据。
2. **展商端权限局限**：本次调研账号为展商端，主办端（/live，含观众管理/报表/系统通知/表单配置等核心功能）未能访问，需要主办方账号补充调研。
3. **参展字段只读**：展馆号、展位号、展位面积均为只读字段，展商无法自行修改，由主办方后台管理。
4. **页面 snapshot 为空**：直接 URL 导航到部分子路由（如 `/ex-live/exhibitor/exhibitorInfo`）会返回空页面，需通过菜单点击触发 Vue Router 渲染。
5. **导出功能未验证**：因展会已结束、数据为空，各模块的导出按钮状态未能充分验证，实际导出能力待主办端调研补充。
